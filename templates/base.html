Welcome to Reef Metrics

Here’s a quick guide to the main areas of the app. Click any card to jump there.

Dashboard
Your tanks, recent readings, and alerts in one place.
Tools
Use the dosing calculator and dose planner to map your additions.
Shortcuts: Dosing Calculator, Dose Planner.
Manage
Set parameters, additives, and test kits that power calculations.
Start with Parameters → Additives → Test Kits.
Dosing Settings
Pick dosing methods, enter daily amounts, and track containers.
Open any tank → Edit Dosing.
Notifications
Check the bell for low-container warnings and actions.
Dismiss alerts or jump to the tank.
Data Center
Import/export and backups for power users.
Export data, restore backups, or bulk import.
Quick Start
  1. Create a tank from the dashboard.
  2. Add parameters, targets, and preferred additives.
  3. Enter initial readings and set dosing methods.
  4. Use the calculator to dial in daily dosing.
