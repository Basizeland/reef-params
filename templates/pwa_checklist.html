PWA QA Checklist

Validate install, offline, and notification behavior across devices.

Install & offline

  • Android Chrome: install prompt appears and app launches in standalone mode.
  • iOS Safari: “Add to Home Screen” flow works and app opens without browser chrome.
  • Offline: navigating to dashboard shows offline page and cached UI loads.

Push notifications

  • Enable push on desktop/Android and confirm status badge shows “On”.
  • Trigger a test alert and verify notification arrives.
  • Disable push and confirm subscriptions are removed.

Performance

  • Lighthouse PWA score ≥ 90 for mobile and desktop.
  • Dashboard load feels responsive; skeleton placeholders appear briefly.
